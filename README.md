# aryane4.github.io

Portfolio site built with Vite + React + TypeScript.

## Deploy to GitHub Pages

1. Go to your repo **Settings → Pages**.
2. Under **Source**, select **Deploy from a branch**.
3. Set **Branch** to `gh-pages / (root)`.
4. Push to `main` — the `.github/workflows/deploy.yml` workflow will build and deploy automatically to the `gh-pages` branch.
